ルーレット 仕様書


============================================================
概要
============================================================

アプリ名:   ルーレット
ジャンル:   演出付きルーレットツール

どんなアプリ?
  カスタム項目を設定して回せるルーレットアプリ。
  様々な演出(紙吹雪、スポットライト、ドラムロール音、スロット風表示等)を搭載。
  罰ゲーム決め、順番決め、ランチ決め、プレゼント抽選など幅広い用途に対応。
  演出の豪華さが最大の特徴。「誰が当たるかわからない」ドキドキ感を最大化する。

誰が使う?
  友達同士、飲み会、イベント司会、教室での決め事など。


============================================================
機能一覧
============================================================

ルーレット本体
  円形ルーレット (Canvas描画)
  各項目が均等に配置され、色分けされる
  スワイプまたはボタンタップで回転開始
  物理演算風の減速 (イージングで自然に停止)
  針(ポインター)が当たった項目が結果

項目の設定
  テキスト入力で項目を追加 (最大20項目)
  項目の削除・並べ替え
  各項目の色を自動割当 (手動変更も可能)
  各項目の当選確率を均等 or 重み付け設定
  プリセットテンプレート
    順番決め (1〜10の番号)
    罰ゲーム (一気飲み/モノマネ/奢る/真実の告白/スキップ)
    ランチ (ラーメン/カレー/寿司/パスタ/ハンバーガー/定食)
    YES/NO
    あみだくじ風 (当たり/ハズレ)

演出 (コア機能)

  回転中の演出
    ドラムロール音 (Web Audio API)
    ルーレットの回転に合わせたティック音
    スピード感のある背景ブラー効果
    項目がポインターを通過するたびにハイライト

  停止直前の演出 (ニアミス演出)
    減速時にわざとゆっくりになる「じらし」演出
    停止寸前のスロー再生 + ハートビート音
    背景の色が当選項目のカラーに徐々に変化

  結果発表の演出 (5種類から選択)
    紙吹雪: 画面全体にカラフルな紙吹雪が降る (Canvas)
    スポットライト: 当選項目にスポットライトが当たる (CSSアニメーション)
    爆発: 中心から光が放射状に広がる (Canvas)
    花火: 画面に花火が打ち上がる (Canvas パーティクル)
    シンプル: 当選項目がバウンスアニメーションで拡大表示

  結果表示
    当選項目名を大きく表示
    ファンファーレ音 (Web Audio API)
    もう一度回すボタン
    当選項目を除外して再度回すオプション

音声設定
  効果音のON/OFF
  ボリューム調整
  効果音の種類選択 (カジノ風 / パーティー風 / シンプル)

履歴
  過去の結果を記録
  同じ項目が連続で出ていないかの確認
  結果の出現回数統計

テーマ
  ダークモード / ライトモード切り替え


============================================================
画面構成
============================================================

メイン画面
  上部: アプリタイトル + 設定ボタン + テーマ切替
  中央: ルーレット本体 (大きく表示)
    円形ルーレット + 上部にポインター(矢印)
    中央にスタートボタン (タップで回転)
  下部: 項目リスト (編集モードトグル)

項目編集パネル (スライドアップ)
  入力フィールド + 追加ボタン
  項目リスト (ドラッグで並替え、スワイプで削除)
  プリセットボタン群
  確率設定トグル (ONで各項目にスライダー表示)

結果オーバーレイ (全画面)
  演出アニメーション
  結果テキスト (大きく中央に)
  もう一度ボタン / この項目を除外して回すボタン / 閉じるボタン

設定画面 (モーダル)
  演出エフェクト選択 (紙吹雪/スポットライト/爆発/花火/シンプル)
  効果音ON/OFF + ボリューム
  効果音スタイル (カジノ/パーティー/シンプル)
  履歴クリアボタン

履歴画面 (モーダル)
  結果一覧 (日時 + 項目名)
  各項目の出現回数バーチャート


============================================================
技術構成
============================================================

フレームワーク: なし (HTML / CSS / JavaScript)
データ永続化:  LocalStorage
描画:          Canvas API (ルーレット、演出エフェクト)
音声:          Web Audio API (効果音)

ファイル構成
  roulette/
    docs/
      仕様書.txt       この仕様書
    index.html          メインHTML
    style.css           スタイルシート
    app.js              アプリロジック
    effects.js          演出エフェクト (紙吹雪/花火/爆発等)
    audio.js            効果音エンジン


============================================================
LocalStorageキー
============================================================

roulette-items       項目データの配列
                     各項目: id, text, color, weight
roulette-settings    設定
                     effect (演出タイプ)
                     soundEnabled, soundVolume, soundStyle
                     theme
roulette-history     結果履歴の配列
                     各結果: text, timestamp


============================================================
ルーレット描画設計
============================================================

Canvas描画
  中心座標を基準に項目数分の扇形を描画
  各扇形にテキストを配置 (フォントサイズは項目数に応じて自動調整)
  回転はCanvas全体のtranslate + rotate で実現

回転アニメーション
  初速をランダム (10〜20 rad/s)
  摩擦係数で毎フレーム減速 (speed *= 0.98)
  速度が閾値以下になったら「じらし演出」に移行
  じらし中はさらにゆっくり減速 (speed *= 0.995)
  完全停止で結果判定

結果判定
  ポインターの角度とルーレットの回転角度から
  どの項目に止まったかを算出
  weight(重み)は各項目の角度幅に反映

演出エフェクト設計 (effects.js)
  紙吹雪: 200個のパーティクルが上から落下 (色/サイズ/角度ランダム)
  花火: 中心から放射状にパーティクルが飛散、重力で落下
  爆発: 光の放射 + 波紋アニメーション
  スポットライト: CSS radial-gradient + opacity アニメーション
  各エフェクトは2〜3秒で自動終了
